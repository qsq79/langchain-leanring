# LangChain 1.x 学习路径指南

欢迎使用LangChain 1.x核心组件学习项目！本指南将帮助您系统地掌握LangChain框架的核心概念和实际应用。

## 学习目标

通过完成本项目的学习，您将能够：
- 深入理解LangChain 1.x的8个核心组件
- 掌握每个组件的基本使用和高级功能
- 了解组件间的协作方式
- 具备实际项目开发能力
- 准备相关技术面试

## 学习路径概览

### 阶段一：基础入门（第1-2周）

#### 第1周：核心概念和基础组件
1. Models组件（01-models/）
   - 阅读Models组件学习指南
   - 运行基础示例
   - 尝试高级示例
   - 学习重点：LLM、Chat Model、Embeddings的区别和使用场景
   - 面试准备：重点掌握Models组件的5个面试题

2. Prompts组件（02-prompts/）
   - 阅读Prompts组件学习指南
   - 运行基础示例
   - 尝试高级示例
   - 学习重点：Prompt Template的设计模式和优化技巧
   - 实践项目：创建一个多场景的Prompt模板库

#### 第2周：工作流构建
3. Chains组件（03-chains/）
   - 阅读Chains组件学习指南
   - 运行基础示例
   - 尝试高级示例
   - 学习重点：不同Chain类型的选择和组合策略
   - 实践项目：构建一个多步骤的文档分析流程

4. Indexes组件（04-indexes/）
   - 阅读Indexes组件学习指南
   - 运行基础示例
   - 尝试高级示例
   - 学习重点：文档处理、向量存储和检索策略
   - 实践项目：实现一个基于文档的问答系统

### 阶段二：进阶应用（第3-4周）

#### 第3周：智能应用开发
5. Memory组件（05-memory/）
   - 阅读Memory组件学习指南
   - 运行基础示例
   - 尝试高级示例
   - 学习重点：不同Memory类型的适用场景和性能优化
   - 实践项目：开发一个具有记忆的对话助手

6. Agents组件（06-agents/）
   - 阅读Agents组件学习指南
   - 运行基础示例
   - 尝试高级示例
   - 学习重点：Agent的工作原理和工具集成
   - 实践项目：构建一个能够使用多种工具的智能助手

#### 第4周：系统优化和扩展
7. Tools组件（07-tools/）
   - 阅读Tools组件学习指南
   - 运行基础示例
   - 尝试高级示例
   - 学习重点：自定义工具开发和性能优化
   - 实践项目：开发一套特定领域的专业工具

8. Callbacks组件（08-callbacks/）
   - 阅读Callbacks组件学习指南
   - 运行基础示例
   - 尝试高级示例
   - 学习重点：监控系统设计和性能优化
   - 实践项目：实现一个完整的监控系统

### 阶段三：综合实战（第5-6周）

#### 第5周：项目实战
9. 综合项目开发
   - 选择一个实际应用场景（如：智能客服、文档分析系统、个人助手等）
   - 设计系统架构，确定使用的组件组合
   - 实现核心功能
   - 集成多个组件
   - 添加监控和错误处理

#### 第6周：优化和部署
10. 项目完善
   - 代码重构和优化
   - 添加测试用例
   - 编写项目文档
   - 部署和演示
   - 收集反馈并改进

## 环境准备

### 基础环境
```bash
# 克隆项目
git clone <repository-url>
cd langchain-learning

# 创建虚拟环境
python -m venv venv
source venv/bin/activate  # Linux/Mac
# 或
venv\Scripts\activate  # Windows

# 安装依赖
pip install -r requirements.txt

# 设置环境变量
export OPENAI_API_KEY="your-openai-api-key"
# 或创建 .env 文件
echo "OPENAI_API_KEY=your-openai-api-key" > .env
```

### 可选环境
```bash
# 安装特定组件的依赖
pip install -r 01-models/requirements.txt
pip install -r 02-prompts/requirements.txt
# ... 其他组件类似

# 开发环境
pip install pytest pytest-asyncio black flake8
```

## 学习方法建议

### 1. 理论学习
- 阅读顺序：先阅读README.md了解概念，再查看代码示例
- 重点标注：在文档中标记重要概念和面试题
- 笔记整理：建立自己的知识体系，记录学习心得

### 2. 实践操作
- 代码运行：每个示例都要亲自运行，观察输出结果
- 参数调整：尝试不同的参数配置，理解其影响
- 错误调试：遇到错误时，分析原因并解决

### 3. 扩展练习
- 功能扩展：在示例基础上添加新功能
- 场景应用：将学到的知识应用到实际场景
- 组件组合：尝试组合不同组件解决复杂问题

```python

def learning_approach():

    # 2. 运行示例
```

5. 调试工具：使用调试器和日志系统
